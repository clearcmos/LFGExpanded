# Changelog

## v1.0.0

**Initial Release:**

- Side panel attached to LFG Browse frame with two-column layout (Groups / Singles)
- Role filters (Tank, Healer, DPS) with include and exclude support
- Class filters (all 9 TBC classes) with include and exclude support
- Left-click to include, right-click to exclude roles and classes
- Show/Hide checkboxes per section to hide all groups or all singles
- Exclusion semantics: groups reject if excluded role/class is present; singles reject only if all listed roles are excluded
- Smart sorting: groups first, then singles sorted by class
- Auto-removes delisted entries in real-time
- Scroll position preservation during filter updates
- Drag-to-resize corner grip for uniform panel scaling
